What is Sevn?

Sevn is a simple todo list application that only shows tasks for the next week. This focuses the user on the tasks that are immediately achievable.

How many ways can I access Sevn?

Sevn will be deployed as a web application built with Ruby on Rails and an iOS application built with Sencha touch and deployed with PhoneGap.

When can I start using Sevn?

Sevn on the web will be ready to begin beta testing in February of 2011.
